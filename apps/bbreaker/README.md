# Brick Breaker

A simple BreakOut clone for the Banglejs

![Screenshot](bbreaker.png)

## Usage

![Screenshot](playing.png)

The "BrickBreaker" moves by itself and you need to use button 1 for Left and 2 for Right.
This behaviour wasn't intended at first but the advantage of using it is we press the buttons less times and makes the game a bit more challenging as you need to calculate when the ball need to touch the BrickBreaker.

## Disclaimer

This game was created to learn JS and how to interact with Banglejs, meaning that it may not be perfect :).
Build with love with base on the tutorial: 2D breakout game using pure JavaScript
https://developer.mozilla.org/en-US/docs/Games/Tutorials/2D_Breakout_game_pure_JavaScript
And with some inspiration from other games in the Banglejs App site.
